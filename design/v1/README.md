# MyiUI Design v1

Open Design daemon was unavailable during Phase 0; this bundle follows the approved wireframe and plan spec.

- **tokens.json** — glass, colors, typography
- **layout.json** — LogoBar / Manager / ProfileCard / NavStack
- **motion.json** — hover/press easing
- **background_spec.json** — video/image cover rules
- **preview/index.html** — interactive HTML reference

Sync to runtime: copy JSON files to `config/menu/` before build or at runtime.
